This report has been written by four software students at Aalborg University as their bachelor project.
In the report \textit{we} refers to the authors of the paper, unless otherwise stated.
We thank the other groups in the multi--project, especially group SW615F16 which we have worked closely with.
We would also like to thank our supervisor Kim A. Jakobsen for his guidance during the semester.

The full code we use is available in our git repositories at \url{http://git.giraf.cs.aau.dk/Giraf/}.
When we finished our progress on GIRAF we made a tag on each branch called \texttt{end2016-*branchname*}.
In Gogs (the git server/frontend used) tags are called releases.
A full list of the hashes for the tags is available in \myref[name]{app:source}.
